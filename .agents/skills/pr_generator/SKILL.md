# PR Material Generator

このスキルは、新しく作成したアプリケーションの魅力を効果的に伝えるためのPR素材を一括で生成するための手順書です。アプリの機能やターゲット層を分析し、以下の素材を自動で作成してください。

## 必須生成アイテムとフォーマット

各アイテムは以下のファイル名とフォーマットでルートディレクトリまたは指定のディレクトリに作成してください。

### 1. PR用ブログ記事 (`blog_pr.md`)
- **形式:** Markdown
- **言語:** 日本語
- **内容:** 
  - キャッチーなタイトル
  - アプリ開発の背景（あるあるや課題の提示）
  - 主な特徴（3つ程度、機能よりもユーザー体験を強調する）
  - おすすめのターゲット層
  - 簡単な使い方のステップ

### 2. サムネイル案 (`thumbnail_ideas.json` & `thumbnail_ideas_en.json`)
- **形式:** JSON
- **言語:** 日本語版と英語版の2ファイルを作成
- **内容:** YouTubeやブログ記事のサムネイル画像を作成するためのプロンプト案（最低3パターン）。
  - 各パターンの要素：`id`, `concept` (コンセプト名), `visuals` (背景、テキスト、UI要素などの視覚的詳細), `colors` (テーマカラー), `target_emotion` (視聴者に与えたい感情)。

### 3. PR動画生成用プロンプト・ナレーション (`video_prompt.json` & `video_prompt_en.json`)
- **形式:** JSON
- **言語:** 日本語版と英語版の2ファイルを作成
- **内容:** 13〜15秒程度のショート動画（TikTok、YouTube Shorts、Instagram Reels用）を作成するための指示書。
  - **重要事項:** 映像内にはテキストを含めないこと（`NO text on screen`）。抽象的なUIや表情の変化で魅力を伝える。
  - 各要素：`video_concept`, `instructions` (秒数、映像スタイル、制約事項), `scenes` (時間帯、`visual_prompt`, `narration_japanese` または `narration_english`), `audio_guidance`。

### 4. YouTube用説明文 (`youtube_description.md`)
- **形式:** Markdown
- **言語:** 日本語と英語を併記（1つのファイルに両方の言語を含める）
- **内容:** 
  - タイトル案
  - アプリの紹介と特徴
  - コールトゥアクション（アプリのリンク挿入用プレースホルダー `[リンク]` や実際のURLなど）
  - ハッシュタグ（5〜6個程度）

### 5. X (旧Twitter) 用ポスト案 (`x_pr_posts.md`)
- **形式:** Markdown
- **言語:** 日本語と英語を併記
- **内容:** 各言語につき、以下の2パターンのポスト案を作成。
  1. 共感・課題解決特化型（日常のペインポイントに寄り添うアプローチ）
  2. 機能紹介特化型（図解や特徴を箇条書きで分かりやすく伝えるアプローチ）
  - どちらもハッシュタグとリンク用のプレースホルダーを含めること。

## 実行時の注意点
- アプリのUIデザイン（色使い、グラスモーフィズムなど）や機能（タグ付け、検索など）をしっかりと素材の内容に反映させること。
- ユーザーから特定のトーン＆マナーの指定がある場合は、それに従うこと。
- PRのターゲット層（例：ビジネスマン、学生、一般ユーザーなど）を明確にしてからテキストを生成すること。
